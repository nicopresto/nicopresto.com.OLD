The dotSkapes Project - open web suite

The dotSkapes Project is a suite of web tools for finding, analyzing, sharing, and tagging global research. The goal is to build global resilience to disease and disaster via an open community platform. Our research is currently being applied to infectious disease outbreaks and analytical capacity during natural disasters. For more information, please visit skapes.org


SaVaGelib - SVG graphing library

My lab group (led by Zack Krejci) is developing an open source SVG graphing library. While most of our graphing needs can be met by Matplotlib and R, we decided to roll our own library for the standard graphs in Healthscapes. We pulled this sub-project from the main project to facilitate it's use by other developers. For access to the code, please visit Launchpad
